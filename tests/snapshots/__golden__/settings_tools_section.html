Settings export / import

Export

Download every freeman_core_* and freeman_digital_* option as a JSON envelope. Runtime state (logs, boot diagnostics, the backup store) is excluded.

Import

Settings import is disabled. Enable with: wp option update freeman_core_tools_settings_import_enabled 1

Backups

Auto-backups created before each import. The five most recent are kept. A restore itself creates a backup, so any restore is also undoable.
