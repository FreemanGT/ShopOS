# Infinite Scroll — Changelog

## 1.0.0
- Initial port from `shopos-infinite-scroll` v1.0.5.
- Adds skeleton count, max pages, and end-message settings via ShopOS admin.
- Debug flag renamed to `?shopos_is_debug=1` (legacy `?shopos_debug=1` still works).
- Front-end CSS classes retained (`shopos-*`) for customer overrides.
